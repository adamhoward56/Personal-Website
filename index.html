About Me

Hi there! I'm an undergrad student studying computer science at the University of Maryland, College Park. I've been programming for nearly a decade and use my free time to develop games, websites, apps, and more. Below are a few of my favorites.

Projects

  • Spiroll

    App Store

    My latest game for iPhone, developed in Unity.

  • RGBY

    App Store / Google Play

    A fast-paced mobile game for iPhone and Android! I wrote RGBY in Lua using CoronaSDK. All promotional graphics were made with Gimp.

  • TinySketch

    Chrome Web Store

    Featured on the front page of the Chrome Web Store with more than 1 million total installs! TinySketch is a drawing tool for Google Chrome.

  • Switchy Bridge

    Google Play

    The first game I published to the Google Play Store. Made with C#, Unity, and Blender! Switchy Bridge has Google Play leaderboard support and video advertisements.

  • Clean Tab

    Chrome Web Store

    I don't like Chrome's new tab page, so I made my own. The extension uses HTML, CSS, and Javascript. It calls the Chrome data-storage API to save user preferences.

  • Parkour Movement Demo

    A first-person parkour movement prototype I built with Unreal Engine.
